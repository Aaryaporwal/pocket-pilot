# PocketPilot – Smart Expense Tracker (C++)

## Description
PocketPilot is a console-based expense tracker written in C++.  
It helps users manage spending across different categories and alerts users when expenses exceed limits.

## Features
- Input pocket money
- Track expenses by category
- Calculate total expenses
- Warning system when limit is crossed
- ASCII expense graph visualization
- Sound alert using Beep()

## Technologies Used
- C++
- Object-Oriented Programming (Classes)
- Arrays
- Windows Beep API

## How to Run
1. Compile using a C++ compiler
2. Run the program
3. Enter expense details
4. View total and warnings
